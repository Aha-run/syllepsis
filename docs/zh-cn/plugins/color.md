# ColorPlugin <!-- {docsify-ignore-all} -->

```typescript
import { ColorPlugin } from '@syllepsis/access-react';

plugins: [
  new ColorPlugin({
    default: string, //  默认字色，值为这个值时会取消样式，默认是#000000
  }),
];
```
